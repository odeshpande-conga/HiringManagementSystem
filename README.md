# Hiring Management System

The Hiring Management System is a web-based application that helps recruiters
manage job postings and candidates apply for jobs online.

## User Roles

### Admin
- Manages users and overall system

### Recruiter
- Creates job openings
- Views applicants
- Updates application status

### Candidate
- Registers and logs in
- Views available jobs
- Applies for jobs
- Tracks application status

## Main Features
- User Authentication (Login/Register)
- Job Posting Management
- Job Search and Viewing
- Apply for Jobs
- Application Tracking
- Recruiter Dashboard
- Candidate Dashboard

## Workflow
1. Candidate registers and logs in
2. Recruiter posts jobs
3. Candidate views jobs and applies
4. Recruiter checks applications
5. Recruiter updates status to one of:
   - Applied
   - Shortlisted
   - Rejected
   - Hired
